Kanap :

Construction d'un site e-commerce en JavaScript

Contexte du projet

Conception d'une plateforme e-commerce dynamique en JavaScript pour la marque Kanap afin d'augementer ses ventes grâce à une boutique en ligne.

Support de soutenance

Ci-dessous, le diaporama utilisé lors de la soutenance de ce projet.

Le site web

Le site web est accessible depuis le bouton ci-dessous.
Afin d'afficher la page, suivez les étapes suivantes :
1. Cliquez sur le bouton
2. Cliquez sur "Code" puis sur "Download ZIP"
3. Extraire du zip et ouvrir dossier sur votre éditeur de code
4. Tapez dans terminal de commande les lignes suivantes: "cd back" puis "node server"
